# Financial-Markets-Intrinsic-Dimensionality-and-Random-Matrix-Structure
The goal of this project is threefold: to find the intrinsic dimensionality of market returns, to identify the  meaningful factors in the covariance matrices of asset returns, and to test if the intrinsic dimensionality  changes across time or market regimes.

Data Sourcing:
For S&P constituency, we used the public GitHub dataset produced by fja05680. The link to which is: https://github.com/fja05680/sp500
